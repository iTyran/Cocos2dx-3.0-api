详细描述

编辑框类

本控件可用于让用户输入少量文本

成员枚举类型说明

enum InputFlag
strong

EditBox::InputFlag定义了被输入文字的显示格式

枚举值
PASSWORD 

表示输入文本为机密数据，只要可能就不要直接显示。 此标志暗含了SENSITIVE标志。

SENSITIVE 

表示输入文字为敏感数据，在实现时禁止为了预测、自动填充 或者其余加速输入的目的而将其存入字典或者表格。 例如信用卡号就是一种敏感数据。

INITIAL_CAPS_WORD 

表示在文本编辑的过程中，每个单词的首字母都应该大写。

INITIAL_CAPS_SENTENCE 

表示在文本编辑的过程中，每句话的首字母都应该大写。

INTIAL_CAPS_ALL_CHARACTERS 

自动将所有字母大写。

enum InputMode
strong

EditBox::InputMode定义了用户可输入的文本类型

virtual void setAnchorPoint ( const Vec2 &  anchorPoint)
overridevirtual

设置锚点，用百分比表示。

一个锚点是所有的转换和定位操作发生的点。 它就像在节点上连接其父类的大头针。 锚点是标准化的，就像百分比一样。(0,0)表示左下角，(1,1)表示右上角。 但是你可以使用比（1,1,）更高的值或者比（0,0）更低的值。 默认的锚点是（0.5,0.5），因此它开始于节点的中心位置。

注解
如果节点有物理体，锚点必须在中心，你不能够这个值。
参数
anchorPoint节点的锚点。
